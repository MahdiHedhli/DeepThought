Safety model

Autonomy is a capability to be bounded, not a personality to be trusted

What it does

  • Maps systems and generates prioritized vulnerability hypotheses
  • Coordinates bounded specialist workers behind a typed envelope firewall
  • Verifies candidates in a sandbox — behind an explicit human stop
  • Preserves durable, portable research memory across sessions and harnesses
  • Prepares SARIF, OSV, CSAF, OpenVEX, and CVE 5.1 artifacts

What it does not do

  • Authorize itself or silently expand its own scope
  • Execute untrusted target code without explicit human approval
  • Transmit a disclosure — drafting belongs to the machine, sending to a person
  • Declare a finding verified without reproducible evidence
  • Explain why developers keep deserializing untrusted input

Unsafe input becomes a controlled refusal, not an accidental adventure.

Early experiments

Preliminary, and labeled as such

DeepThought runs early research programs. The vuln-rediscovery skill builds a static detector for a vulnerability class — calibrated on one seed CVE, graded on held-out CVEs it never saw, using real source at pinned vulnerable and patched SHAs. Twelve classes are measured today at a preliminary ~79% mean held-out generalization, versioned under a regression bar. XXE sits at an honest 1/3 — reported, not hidden.

These are early experiments. The numbers, the honest curve, and a rough cross-model field note are preliminary — small internal corpora, directional signals, not settled benchmarks. The full detail and every caveat live in the experiments log.

Get started

The Guide, in three verbs

Install, register an authorized target, validate the store, and prepare local artifacts. Nothing is transmitted.

# install
git clone https://github.com/MahdiHedhli/DeepThought.git
cd DeepThought
uv venv --python 3.12 .venv
uv pip install --python .venv -e ".[dev]"

# run a typed research session
.venv/bin/deepthought playbook new-project --name "PHP src" \
  --git-url https://github.com/php/php-src --basis permissive_oss --scope ext/soap

# validate state, schemas, lifecycle, conformance
.venv/bin/deepthought check

# emit prepared local artifacts — transmit nothing
.venv/bin/deepthought publish
